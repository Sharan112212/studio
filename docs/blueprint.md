# **App Name**: FolioForge

## Core Features:

- Profile Showcase: Display the user's name, a brief bio, and a professional headshot.
- Project Gallery: Showcase the user's projects with descriptions, images, and links to live demos or code repositories.
- Skills Highlight: List the user's key skills and areas of expertise.
- Basic Contact: Display icons that link to the user's GitHub, Gmail, and LinkedIn profiles.

## Style Guidelines:

- Primary color: Deep navy blue (#2E3A59) for a professional and trustworthy feel.
- Background color: Light gray (#F0F4F8) to provide a clean backdrop.
- Accent color: Teal (#4DB6AC) to highlight key elements and calls to action.
- Body and headline font: 'Inter', a grotesque-style sans-serif, for a modern and neutral appearance suitable for both headlines and body text.
- Use simple, minimalist icons to represent skills and technologies.
- Employ a clean, single-page layout with clear sections and intuitive navigation.
- Incorporate subtle transitions and animations to enhance user experience without being distracting.
